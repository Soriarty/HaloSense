# DFRobot C4001 Official Datasheets

This directory contains the official PDF datasheets and technical documentation from DFRobot.

## Files

- **dfrobot-c4001-dimensional-drawing.pdf** - Dimensional Drawing
  - Source: https://img.dfrobot.com.cn/wiki/5ea64bf6cf1d8c7738ad2881/35d327648fb4b29373af27a13d9d7405.pdf
  - Contains: Physical dimensions, mechanical drawings, mounting specifications

- **dfrobot-c4001-communication-protocol.pdf** - Communication Protocol
  - Source: https://dfimg.dfrobot.com/5ea64bf6cf1d8c7738ad2881/wiki/3b88a4ecd7d7f18918a0fa8ba1f970c6.pdf
  - Contains: UART command structure, protocol specifications, message formats

## Online Resources

- **Product Wiki:** https://wiki.dfrobot.com/SKU_SEN0609_C4001_mmWave_Presence_Sensor_25m
- **Product Page:** https://www.dfrobot.com/product-2639.html

## Usage

These PDF files are reference materials for:
- **Dimensional Drawing:** PCB layout planning, enclosure design, mounting hole positions
- **Communication Protocol:** UART command implementation, custom firmware development, deep integration

For quick reference and general specifications, see the main sensor documentation: [../README.md](../README.md)
