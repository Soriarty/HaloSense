## Usage

These PDF files are reference materials for:
- **Module Datasheet:** Pin assignments, electrical specs, antenna design, mechanical dimensions
- **SoC Datasheet:** Chip capabilities, peripheral features, memory maps
- **Hardware Guidelines:** PCB layout, RF design best practices, component selection
- **Technical Manual:** Register-level programming, peripheral configuration, timing diagrams
